Input Format Reference

🔖 Markdown

===
# Optional topic heading
Front text of the card
here on multiple lines
---
Back text / answer
goes here
===
# Next Card
Another question
---
Another answer
===

The === separator can be changed above.
Front / back split with a --- on its own line.

{ } JSON

[
  {
    "topic": "Array Access",
    "front": "Question text",
    "back": "Answer text"
  },
  {
    "front": "Q without topic",
    "back": "Answer"
  }
]

Aliases: question/q → front, answer/a → back.

⚡ YAML

- topic: Array Access
  front: |
    Question text
    multiline ok
  back: |
    Answer text
    also multiline

- front: Simple question
  back: Simple answer

Use | for literal multiline strings. Same field aliases as JSON.
